# Qwen3-VL Technical Report

Source:

- Paper: `references/papers/qwen3_vl_technical_report_2511.21631.pdf`
- URL: `https://arxiv.org/abs/2511.21631`

QTRM relevance:

- Prior vision-language architecture reference in the Qwen family.
- Useful for visual tokenization, image/video reasoning, OCR, grounding, and
  long-context multimodal evaluation.

Relationship to donor:

- `Qwen/Qwen3.5-2B-Base` is a newer native vision-language foundation model.
- Qwen3-VL is not the direct donor, but it is relevant for MemoryOS visual
  retrieval/evaluation design.
